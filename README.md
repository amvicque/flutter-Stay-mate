# StayMate 🏡 – Your Smart Stay Companion

StayMate is a Flutter-based mobile application that allows users to browse and book short-term rental properties, just like Airbnb. It bridges the gap between travelers and property owners looking to offer accommodation on flexible terms.

## 🌍 SDG Alignment
StayMate supports **SDG 11: Sustainable Cities and Communities** by promoting affordable and accessible housing solutions for travelers and residents.

---

## ✨ Features
- 🔍 Browse listings by location, price, and availability
- 🗓️ Book accommodations for specific dates
- 👤 User roles: Host and Guest
- 📝 Reviews and ratings system
- 🧭 Optional: Map integration for property location view
